# Boxing Analyst - Powered by Groq Compound Beta

An AI-powered boxing intelligence platform for The Ring partnership using Groq's Compound Beta.


## Project Structure

```
src/
├── app/
│   ├── api/                 # Next.js API routes
│   │   ├── analyze/         # Fighter profile generation
│   │   ├── search/          # Fighter search
│   │   └── compound/        # Q&A endpoints
│   ├── components/          # React UI components
│   │   ├── LandingPage.tsx  # Hero page with branding
│   │   ├── ChatView.tsx     # Main chat interface
│   │   ├── ChatMessage.tsx  # Message bubbles
│   │   ├── FighterProfile.tsx # Fighter cards
│   │   └── ...
│   ├── globals.css          # Global styles & theme
│   └── page.tsx             # Root page orchestrator
├── hooks/
│   └── useChat.ts           # Chat state management hook
├── lib/
│   ├── compound.ts          # Groq API integration
│   ├── types.ts             # TypeScript interfaces
│   ├── constants.ts         # App constants
│   └── utils.ts             # Helper functions
└── public/                  # Static assets (logos, images)
```

## Features

- **Fighter Search**: AI-powered boxer search and matching
- **Fighter Profiles**: Complete profiles with stats, records, and biography
- **Fight History**: Comprehensive fight records and results
- **AI Analysis**: Tactical insights, strengths, weaknesses, and matchup analysis
- **Q&A**: Ask questions about any fighter or general boxing topics
- **Smart Caching**: Client-side profile and answer caching for faster responses
- **Retry Logic**: Automatic retries with exponential backoff for reliability

## Setup

1. Clone the repository.

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create a `.env.local` file:
   ```bash
   touch .env.local
   ```

4. Add your Groq API key:
   ```env
   GROQ_API_KEY=your_groq_api_key_here
   ```
   - Get your key from [Groq Cloud Console](https://console.groq.com/)
   - Ensure you have access to the Compound Beta model

5. Run the development server:
   ```bash
   npm run dev
   ```

6. Open [http://localhost:3000](http://localhost:3000).

- `GROQ_API_KEY`: Your Groq API key (required)
